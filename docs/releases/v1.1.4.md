# Token Meter 1.1.4

Codexの長期利用履歴をより確実に取り込み、Widgetとダッシュボードの表示を改善するアップデートです。更新が見つかったときの通知も、メニューバーアプリとして見落としにくくしました。

## 変更内容

- 未取り込みの古いCodexセッションを初回更新時に補完し、30日表示を含む履歴の欠落を修正
- Widgetの7日間推移を棒グラフから折れ線グラフへ変更
- Widgetの残量バーを安定して描画する実装へ変更
- 更新が見つかったとき、Sparkleの更新画面を最前面に表示
- Token breakdownでCodex行が中央へずれる問題を修正
- Daily tokensの日付ラベルを日だけにして、月表示の見切れを解消
- Menu bar設定をSetupへ一本化し、Settings内の重複項目を削除
- Aboutページの製品画像を高解像度化し、実際のWidgetスクリーンショットへ更新

## 動作環境

- macOS 14.0以降
- Apple Silicon / Intel Mac（Universal Binary）

## インストール

`TokenMeter-1.1.4.zip`を展開し、`TokenMeter.app`を`/Applications`へ移動してください。既にToken Meterを使用している場合は、アプリ内の「Check for Updates…」から更新できます。

アプリはDeveloper IDで署名し、AppleのNotary Serviceで公証しています。更新ZIPはSparkleのEdDSA署名でも検証されます。
