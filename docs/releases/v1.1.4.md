# Token Meter 1.1.4

## English

This update ensures longer-term Codex usage history ingestion and improves widget and dashboard displays. It also makes update notifications harder to miss as a menu bar app.

### Changes

- Fills in older untracked Codex sessions on the first update to fix historical gaps (including 30-day views).
- Changed the Widget's 7-day trend chart from a bar chart to a line chart.
- Updated Widget remaining bar drawing implementation for more stability.
- Brings Sparkle's update window to the foreground when an update is found.
- Fixed a layout issue where the Codex row in the Token Breakdown was shifted to the center.
- Simplified date labels in "Daily tokens" to show only the day of the month, preventing text truncation.
- Consolidated Menu Bar settings into Setup, removing duplicate controls in Settings.
- Replaced the product screenshots on the About page with high-resolution ones using actual widgets.

### System Requirements

- macOS 14.0 or later
- Apple Silicon / Intel Mac (Universal Binary)

### Installation

Unzip `TokenMeter-1.1.4.zip` and move `TokenMeter.app` to `/Applications`. If you are already using Token Meter, you can update from the menu by clicking "Check for Updates…".

The app is Developer ID signed and notarized by Apple. The update ZIP is also verified by Sparkle's EdDSA signature.

- Token breakdown에서 Codex 행이 중앙으로 어긋나던 버그 수정.
- Daily tokens의 날짜 라벨을 일(day)만 표시하도록 하여 월 표시가 잘리는 현상 해결.
- 메뉴 막대 설정을 Setup 가이드로 일원화하고 Settings 창의 중복 항목 제거.
- 소개(About) 페이지의 제품 이미지를 고해상도화하고 실제 위젯 스크린샷으로 교체.
