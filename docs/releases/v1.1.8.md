# Token Meter 1.1.8

表示と言語対応を磨き、利用枠の状態と更新時刻をより自然に確認できるようにするアップデートです。

## 変更内容

- アプリで選択した言語に合わせて、日付・時刻・相対時刻を表示。システムの12/24時間表示設定も維持
- 残量、最終更新、リセット時刻、通知メッセージを日本語・中国語（簡体）・韓国語に対応
- 利用枠のリセット通知に「5時間」「週次」など対象の枠を表示し、内容を分かりやすく改善
- ダッシュボードの日別グラフの軸ラベルをデータ点と正確に揃えて表示
- Widgetの曜日表示をアプリの選択言語に合わせ、残量表示とアクセシビリティ用文字列を改善
- 設定・初期設定画面のOAuthエンドポイント表記と日本語表現を見直し

## 動作環境

- macOS 14.0以降
- Apple Silicon / Intel Mac（Universal Binary）

## インストール

`TokenMeter-1.1.8.zip`を展開し、`TokenMeter.app`を`/Applications`へ移動してください。既にToken Meterを使用している場合は、アプリ内の「Check for Updates…」から更新できます。

アプリはDeveloper IDで署名し、AppleのNotary Serviceで公証しています。更新ZIPはSparkleのEdDSA署名でも検証されます。
