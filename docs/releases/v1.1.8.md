# Token Meter 1.1.8

## English

This update refines display and language support, making it easier and more natural to check usage limit status and update times.

### Changes

- Dates, times, and relative times are now displayed according to the selected app language, while preserving system 12/24-hour clock preferences.
- Remaining usage, last update, reset times, and notification messages now support English, Japanese, Chinese (Simplified), and Korean.
- Reset notifications now include target limit types (e.g., "5-hour" or "weekly") for better clarity.
- Axis labels on the dashboard daily charts are now precisely aligned with data points.
- Widget day-of-week displays now respect the selected app language, and remaining usage displays and accessibility labels have been improved.
- Refined OAuth endpoint labeling and phrasing in Settings and Setup screens.

### System Requirements

- macOS 14.0 or later
- Apple Silicon / Intel Mac (Universal Binary)

### Installation

Unzip `TokenMeter-1.1.8.zip` and move `TokenMeter.app` to `/Applications`. If you are already using Token Meter, you can update from the menu by clicking "Check for Updates…".

The app is Developer ID signed and notarized by Apple. The update ZIP is also verified by Sparkle's EdDSA signature.

## 中文

此更新优化了显示和多语言支持，使查看用量限制状态和更新时间更加自然。

### 变更内容

- 根据应用选择的语言显示日期、时间和相对时间，同时保留系统的 12/24 小时制设置。
- 剩余用量、最后更新时间、重置时间及通知消息已支持中文（简体）、英语、日语和韩语。
- 限额重置通知现在会显示具体的限额类型（如“5小时”或“每周”），以提高清晰度。
- 仪表盘日图表的轴标签现在与数据点精准对齐。
- 小组件的星期显示现已适配应用的选择语言，并改善了剩余用量显示与无障碍标签。
- 优化了设置和初始设置界面中的 OAuth 端点表述。

### 运行环境

- macOS 14.0 或更高版本
- Apple Silicon / Intel Mac（通用二进制）

### 安装方法

解压 `TokenMeter-1.1.8.zip`，然后将 `TokenMeter.app` 移至 `/Applications`。如果您已在使用 Token Meter，可通过应用内的“Check for Updates…”进行更新。

应用已使用 Developer ID 签名并已通过 Apple 公证。更新 ZIP 包也通过 Sparkle 的 EdDSA 签名进行验证。

## 한국어

표시 및 다국어 지원을 개선하여 사용 한도 상태와 업데이트 시각을 보다 자연스럽게 확인할 수 있도록 한 업데이트입니다.

### 변경 내용

- 앱에서 선택한 언어에 맞춰 날짜, 시간, 상대 시간을 표시하며 시스템의 12/24시간 형식 설정도 유지합니다.
- 잔여량, 최근 업데이트, 초기화 시간, 알림 메시지가 한국어, 영어, 일본어, 중국어(간체)를 지원합니다.
- 사용 한도 초기화 알림에 '5시간', '주간' 등 대상 한도를 표시하여 내용을 알기 쉽게 개선했습니다.
- 대시보드 일별 그래프의 축 레이블이 데이터 포인트와 정확히 정렬되도록 수정했습니다.
- 위젯의 요일 표시가 앱의 선택 언어에 맞춰지며, 잔여량 표시 및 접근성 텍스트를 개선했습니다.
- 설정 및 초기 설정 화면의 OAuth 엔드포인트 표기 및 문구를 검토하고 수정했습니다.

### 작동 환경

- macOS 14.0 이상
- Apple Silicon / Intel Mac (Universal Binary)

### 설치 방법

`TokenMeter-1.1.8.zip` 압축을 풀고 `TokenMeter.app`을 `/Applications`로 이동하십시오. 이미 Token Meter를 사용 중인 경우 앱 내 'Check for Updates…'를 통해 업데이트할 수 있습니다.

앱은 Developer ID로 서명되었으며 Apple의 공증(Notarization)을 받았습니다. 업데이트용 ZIP은 Sparkle의 EdDSA 서명으로도 검증됩니다.
