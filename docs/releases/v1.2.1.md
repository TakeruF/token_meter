# Token Meter 1.2.1

## English

This update fixes Japanese units appearing in the English interface, makes the 90-day chart readable, and lets Japanese, Chinese and Korean users choose how token counts are abbreviated.

### Changes

- Fixed the dashboard charts labelling their axes in the system language instead of the one you picked. On a Japanese Mac with the English interface selected, the charts read "6億" and "16日" where they should have read "600M" and "16".
- The 90-day chart now labels every 10th day instead of every day, so the dates no longer collide into an unreadable smear, and it names the month — a bare "6" says little across three months.
- Added a number format setting for Japanese, Chinese and Korean: token counts can now read as either K/M/B or in myriad units (万 / 億). The choice applies everywhere at once — charts, cards, menu bar and widget — so no two places disagree. K/M/B stays the default.

### System Requirements

- macOS 14.0 or later
- Apple Silicon / Intel Mac (Universal Binary)

### Installation

Unzip `TokenMeter-1.2.1.zip` and move `TokenMeter.app` to `/Applications`. If you are already using Token Meter, you can update from the menu by clicking "Check for Updates…".

The app is Developer ID signed and notarized by Apple. The update ZIP is also verified by Sparkle's EdDSA signature.

## 日本語

英語UIに日本語の単位が混ざる問題の修正、90日グラフの可読性の改善、日中韓でのトークン数の表記選択に対応するアップデートです。

### 変更内容

- ダッシュボードのグラフが、選択した言語ではなくシステムの言語で軸ラベルを表示していた問題を修正。日本語環境のMacで英語UIを選んでいると、「600M」「16」と表示されるべきところが「6億」「16日」になっていました
- 90日グラフのX軸ラベルを、毎日ではなく10日おきに表示するように変更。日付が重なって読めなくなる問題を解消し、あわせて月も表示するように（3か月にまたがると「6」だけでは何月か分かりません）
- 日中韓向けに数値表記の設定を追加。トークン数をK/M/Bと万進法（万 / 億）から選べます。グラフ・カード・メニューバー・ウィジェットのすべてに同時に適用されるため、表示がばらつきません。デフォルトはK/M/Bです

### 動作環境

- macOS 14.0以降
- Apple Silicon / Intel Mac（Universal Binary）

### インストール

`TokenMeter-1.2.1.zip`を展開し、`TokenMeter.app`を`/Applications`へ移動してください。既にToken Meterを使用している場合は、アプリ内の「Check for Updates…」から更新できます。

アプリはDeveloper IDで署名し、AppleのNotary Serviceで公証しています。更新ZIPはSparkleのEdDSA署名でも検証されます。

## 中文

此更新修复了英文界面中混入日文单位的问题，改善了 90 天图表的可读性，并支持中日韩用户选择 Token 数量的表示方式。

### 变更内容

- 修复仪表盘图表使用系统语言而非所选语言标注坐标轴的问题。在日语环境的 Mac 上选择英文界面时，本应显示为“600M”“16”的位置却显示为“6億”“16日”。
- 90 天图表的 X 轴标签改为每 10 天显示一次，而非每天都显示，解决了日期重叠难以辨认的问题，同时增加了月份显示（跨越三个月时，仅凭“6”无法判断月份）。
- 为中日韩新增数字格式设置。Token 数量可在 K/M/B 与万进制（万 / 亿）之间选择。该设置会同时应用于图表、卡片、菜单栏和小组件，因此各处显示不会不一致。默认为 K/M/B。

### 运行环境

- macOS 14.0 或更高版本
- Apple Silicon / Intel Mac（通用二进制）

### 安装方法

解压 `TokenMeter-1.2.1.zip`，然后将 `TokenMeter.app` 移至 `/Applications`。如果您已在使用 Token Meter，可通过应用内的“Check for Updates…”进行更新。

应用已使用 Developer ID 签名并已通过 Apple 公证。更新 ZIP 包也通过 Sparkle 的 EdDSA 签名进行验证。

## 한국어

영어 UI에 일본어 단위가 섞이는 문제를 수정하고, 90일 차트의 가독성을 개선하며, 한중일 사용자가 토큰 수 표기 방식을 선택할 수 있도록 하는 업데이트입니다.

### 변경 내용

- 대시보드 차트가 선택한 언어가 아닌 시스템 언어로 축 레이블을 표시하던 문제를 수정. 일본어 환경의 Mac에서 영어 UI를 선택하면 "600M", "16"으로 표시되어야 할 부분이 "6億", "16日"로 표시되었습니다.
- 90일 차트의 X축 레이블을 매일이 아닌 10일 간격으로 표시하도록 변경. 날짜가 겹쳐 읽을 수 없던 문제를 해소하고, 함께 월도 표시하도록 개선했습니다(3개월에 걸치면 "6"만으로는 어느 달인지 알 수 없습니다).
- 한중일용 숫자 표기 설정을 추가. 토큰 수를 K/M/B와 만진법(만 / 억) 중에서 선택할 수 있습니다. 차트, 카드, 메뉴 막대, 위젯 모두에 동시에 적용되므로 표시가 서로 어긋나지 않습니다. 기본값은 K/M/B입니다.

### 작동 환경

- macOS 14.0 이상
- Apple Silicon / Intel Mac (Universal Binary)

### 설치 방법

`TokenMeter-1.2.1.zip` 압축을 풀고 `TokenMeter.app`을 `/Applications`로 이동하십시오. 이미 Token Meter를 사용 중인 경우 앱 내 'Check for Updates…'를 통해 업데이트할 수 있습니다.

앱은 Developer ID로 서명되었으며 Apple의 공증(Notarization)을 받았습니다. 업데이트용 ZIP은 Sparkle의 EdDSA 서명으로도 검증됩니다.
