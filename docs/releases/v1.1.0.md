# Token Meter 1.1.0

初回起動を分かりやすくし、メニューバーのコンパクト表示をブランドロゴ中心に改善するアップデートです。

## 変更内容

- 4ステップのオンボーディングを追加
  - Token Meterの簡単な紹介
  - Claude／Codexの表示対象を選択
  - メニューバー表示と表示形式を選択
  - 選択内容を確認してDashboardを開始
- 初回起動時にウィンドウが表示されない場合のフォールバックを追加
- Dashboard／Settingsを開いたとき、既存ウィンドウを最前面へ移動する動作を改善
- メニューバーのCompact表示を`C`／`X`からClaude／Codexのブランドロゴへ変更
- SettingsとSetupのプレビューも実際のメニューバー表示に合わせて更新
- オンボーディング各画面のスクリーンショットをドキュメントへ追加

## 動作環境

- macOS 14.0以降
- Apple Silicon / Intel Mac（Universal Binary）

## インストール

`TokenMeter-1.1.0.zip`を展開し、`TokenMeter.app`を`/Applications`へ移動してください。既にToken Meterを使用している場合は、アプリ内の「Check for Updates…」から更新できます。

アプリはDeveloper IDで署名し、AppleのNotary Serviceで公証しています。更新ZIPはSparkleのEdDSA署名でも検証されます。
