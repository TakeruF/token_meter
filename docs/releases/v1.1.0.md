# Token Meter 1.1.0

## English

This update clarifies the first-run experience and improves the menu bar's Compact mode by focusing on brand logos.

### Changes

- Added a 4-step onboarding flow:
  - Brief introduction to Token Meter
  - Select providers to display (Claude / Codex)
  - Choose menu bar items and layout styles
  - Confirm settings and open the Dashboard
- Added a fallback for when the main window fails to display on first run.
- Improved window behavior to reliably bring existing windows to the foreground when opening the Dashboard or Settings.
- Changed the menu bar Compact text from `C` / `X` to the Claude and Codex brand logos.
- Updated previews in Settings and Setup to match the actual menu bar appearance.
- Added screenshots of onboarding screens to the documentation.

### System Requirements

- macOS 14.0 or later
- Apple Silicon / Intel Mac (Universal Binary)

### Installation

Unzip `TokenMeter-1.1.0.zip` and move `TokenMeter.app` to `/Applications`. If you are already using Token Meter, you can update from the menu by clicking "Check for Updates…".

The app is Developer ID signed and notarized by Apple. The update ZIP is also verified by Sparkle's EdDSA signature.

  - 选择菜单栏显示及展现形式

- 将各初始引导界面的截图添加到了项目文档中。

- macOS 14.0 或更高版本
- Apple Silicon / Intel Mac（通用二进制）

### 安装方法

解压 `TokenMeter-1.1.0.zip`，然后将 `TokenMeter.app` 移至 `/Applications`。如果您已在使用 Token Meter，可通过应用内的“Check for Updates…”进行更新。

应用已使用 Developer ID 签名并已通过 Apple 公证。更新 ZIP 包也通过 Sparkle 的 EdDSA 签名进行验证。

## 한국어

최초 실행 과정을 보다 명확하게 가이드하고 메뉴 막대의 컴팩트 표시 방식을 브랜드 로고 위주로 개선하는 업데이트입니다.

### 변경 내용

- 4단계 온보딩(Onboarding) 프로세스 추가:
  - Token Meter의 간단한 기능 소개
  - 표시할 AI 제공자 선택 (Claude / Codex)
  - 메뉴 막대 노출 및 레이아웃 스타일 설정
  - 설정값 확인 후 대시보드(Dashboard) 시작
- 최초 실행 시 창이 표시되지 않는 경우의 예외 처리 추가.
- 대시보드 또는 설정을 열었을 때 기존에 열려 있는 창이 정상적으로 최전면으로 오도록 윈도우 포커스 제어 동작 개선.
- 메뉴 막대의 컴팩트 모드 텍스트 표시를 `C` / `X`에서 Claude 및 Codex 브랜드 로고 이미지로 변경.
- 설정(Settings) 및 Setup 가이드의 미리보기를 실제 메뉴 막대 표시 상태에 맞춰 갱신.
- 온보딩 각 단계 화면의 스크란샷을 문서에 추가.

### 작동 환경

- macOS 14.0 이상
- Apple Silicon / Intel Mac (Universal Binary)

### 설치 방법

`TokenMeter-1.1.0.zip` 압축을 풀고 `TokenMeter.app`을 `/Applications`로 이동하십시오. 이미 Token Meter를 사용 중인 경우 앱 내 'Check for Updates…'를 통해 업데이트할 수 있습니다.

앱은 Developer ID로 서명되었으며 Apple의 공증(Notarization)을 받았습니다. 업데이트용 ZIP은 Sparkle의 EdDSA 서명으로도 검증됩니다.
