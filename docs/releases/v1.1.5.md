# Token Meter 1.1.5

アプリ本体とWidgetを日本語・英語・簡体字中国語・韓国語の4言語に対応させるアップデートです。初回起動中も、設定後も、アプリ内からいつでも表示言語を切り替えられます。

## 変更内容

- アプリ全体を日本語・英語・簡体字中国語・韓国語に対応
- Onboardingの全画面に言語選択を追加
- Settingsの先頭に言語設定を追加
- 選択した言語をメニューバー、通知、接続状態、診断表示へ反映
- 選択した言語をWidgetへ引き継ぎ、Widget内のラベルも4言語化
- メニューバーの残量表示を5時間枠（既定）または週次枠から選択可能にし、5時間枠が提供されていないプロバイダーは週次枠へ自動フォールバック
- Widgetスナップショットの書き込み順を保証し、古いCodex残量が最新値を上書きする競合を修正
- 最小ウィンドウサイズで4言語のOnboarding全4画面とSettingsを検証し、長い文言の折返しと文字切れを調整
- Aboutページのダウンロード先と4言語対応の案内を更新

## 動作環境

- macOS 14.0以降
- Apple Silicon / Intel Mac（Universal Binary）

## インストール

`TokenMeter-1.1.5.zip`を展開し、`TokenMeter.app`を`/Applications`へ移動してください。既にToken Meterを使用している場合は、アプリ内の「Check for Updates…」から更新できます。

アプリはDeveloper IDで署名し、AppleのNotary Serviceで公証しています。更新ZIPはSparkleのEdDSA署名でも検証されます。
