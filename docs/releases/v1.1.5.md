# Token Meter 1.1.5

## English

This update introduces full support for four languages (English, Japanese, Simplified Chinese, and Korean) across the main app and widgets. You can switch languages anytime from within the app, both during first launch and from Settings.

### Changes

- Added full support for English, Japanese, Simplified Chinese, and Korean across the entire app.
- Added a language selector to all onboarding screens.
- Added language settings to the top of the Settings view.
- Reflected the selected language in the menu bar, notifications, connection statuses, and diagnostics.
- Propagated the selected language to widgets, enabling multilingual labels within widgets.
- Allowed choosing between the 5-hour limit (default) or weekly limit for the menu bar display (automatically falls back to weekly if a 5-hour limit is not provided).
- Ensured widget snapshot write ordering, fixing a race condition where older Codex usage data could overwrite the newest value.
- Validated all onboarding and Settings screens at the minimum window size to adjust wrapping and truncation for longer localized strings.
- Updated download links and language support information on the About page.

### System Requirements

- macOS 14.0 or later
- Apple Silicon / Intel Mac (Universal Binary)

### Installation

Unzip `TokenMeter-1.1.5.zip` and move `TokenMeter.app` to `/Applications`. If you are already using Token Meter, you can update from the menu by clicking "Check for Updates…".

The app is Developer ID signed and notarized by Apple. The update ZIP is also verified by Sparkle's EdDSA signature.

## 中文

此更新使应用本体和小组件支持中文（简体）、英语、日语、韩语 4 种语言。无论是首次启动还是设置后，都可以随时在应用内切换语言。

### 变更内容

- 应用整体支持中文（简体）、英语、日语、韩语。
- 在初始引导 (Onboarding) 的所有界面中添加了语言选择。
- 在设置 (Settings) 顶部添加了语言设置。
- 将选择的语言应用到菜单栏、通知、连接状态和诊断显示。
- 将选择的语言传递给小组件，使小组件内的标签也实现 4 语言化。
- 允许选择菜单栏的剩余用量显示为 5 小时限额（默认）或每周限额（未提供 5 小时限额的提供商将自动退回到每周限额）。
- 保证了小组件快照的写入顺序，修复了旧的 Codex 剩余用量覆盖最新值的竞争条件。
- 在最小窗口尺寸下验证了 4 种语言的所有初始引导界面和设置界面，调整了长文本的换行和截断问题。
- 更新了关于页面的下载链接以及 4 语言支持指南。

### 运行环境

- macOS 14.0 或更高版本
- Apple Silicon / Intel Mac（通用二进制）

### 安装方法

解压 `TokenMeter-1.1.5.zip`，然后将 `TokenMeter.app` 移至 `/Applications`。如果您已在使用 Token Meter，可通过应用内的“Check for Updates…”进行更新。

应用已使用 Developer ID 签名并已通过 Apple 公证。更新 ZIP 包也通过 Sparkle 的 EdDSA 签名进行验证。

## 한국어

앱 본체와 위젯을 한국어, 영어, 중국어(간체), 일본어 4개 언어로 대응하는 업데이트입니다. 최초 실행 시에도, 설정 완료 후에도 앱 내에서 언제든지 표시 언어를 전환할 수 있습니다.

### 변경 내용

- 앱 전체에 한국어, 영어, 중국어(간체), 일본어 대응.
- 온보딩(Onboarding)의 모든 화면에 언어 선택 메뉴 추가.
- 설정(Settings) 상단에 언어 설정 메뉴 추가.
- 선택한 언어를 메뉴 막대, 알림, 연결 상태, 진단 화면에 반영.
- 선택한 언어를 위젯으로 전달하여 위젯 내 라벨도 4개 언어로 적용.
- 메뉴 막대의 잔여량 표시를 5시간 제한(기본) 또는 주간 제한 중에서 선택 가능하게 개선(5시간 제한을 제공하지 않는 제공자는 주간 제한으로 자동 폴백).
- 위젯 스냅샷의 쓰기 순서를 보장하여 오래된 Codex 잔여량이 최신 값을 덮어쓰는 동시성 버그 수정.
- 최소 창 크기에서 4개 언어의 온보딩 화면과 설정을 검증하고 긴 텍스트의 줄바꿈 및 잘림 현상 조정.
- 소개(About) 페이지의 다운로드 경로 및 4개 언어 지원 안내를 업데이트.

### 작동 환경

- macOS 14.0 이상
- Apple Silicon / Intel Mac (Universal Binary)

### 설치 방법

`TokenMeter-1.1.5.zip` 압축을 풀고 `TokenMeter.app`을 `/Applications`로 이동하십시오. 이미 Token Meter를 사용 중인 경우 앱 내 'Check for Updates…'를 통해 업데이트할 수 있습니다.

앱은 Developer ID로 서명되었으며 Apple의 공증(Notarization)을 받았습니다. 업데이트용 ZIP은 Sparkle의 EdDSA 서명으로도 검증됩니다.
