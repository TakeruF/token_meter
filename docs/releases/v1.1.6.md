# Token Meter 1.1.6

ダッシュボードとWidgetのグラフを見やすく刷新するアップデートです。日々の推移を折れ線と棒グラフで確認しやすくしました。

## 変更内容

- ダッシュボードの「Daily tokens」を棒グラフから折れ線グラフに変更し、プロバイダーごとに色分け
- 折れ線の各点にカーソルを合わせると、その日の日付とプロバイダー別トークン数をツールチップ表示
- Largeウィジェットの7日間スパークラインを、軸ラベル付きの棒グラフに刷新
- Settingsの言語選択をラベルと右揃えのメニューに整えて視認性を改善

## 動作環境

- macOS 14.0以降
- Apple Silicon / Intel Mac（Universal Binary）

## インストール

`TokenMeter-1.1.6.zip`を展開し、`TokenMeter.app`を`/Applications`へ移動してください。既にToken Meterを使用している場合は、アプリ内の「Check for Updates…」から更新できます。

アプリはDeveloper IDで署名し、AppleのNotary Serviceで公証しています。更新ZIPはSparkleのEdDSA署名でも検証されます。
