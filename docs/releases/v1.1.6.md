# Token Meter 1.1.6

## English

This update refreshes dashboard and widget charts to make daily trends easier to track with line and bar charts.

### Changes

- Changed "Daily tokens" on the dashboard from a bar chart to a line chart, color-coded by provider.
- Hovering over points on the line chart now displays the date and token counts by provider in a tooltip.
- Refreshed the Large widget's 7-day sparkline into a bar chart with axis labels.
- Aligned the language selector in Settings with labels and right-aligned menus for better visibility.

### System Requirements

- macOS 14.0 or later
- Apple Silicon / Intel Mac (Universal Binary)

### Installation

Unzip `TokenMeter-1.1.6.zip` and move `TokenMeter.app` to `/Applications`. If you are already using Token Meter, you can update from the menu by clicking "Check for Updates…".

The app is Developer ID signed and notarized by Apple. The update ZIP is also verified by Sparkle's EdDSA signature.
