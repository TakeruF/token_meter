# Token Meter 1.1.7

## English

This update significantly improves the readability of the dashboard and widgets, making "actually processed tokens" visible at a glance.

### Changes

- Unified dashboard metrics to "actually processed tokens" (excluding cached content resent each turn). Summary cards now emphasize the processed amount, with the total (including cache) shown alongside.
- Consolidated "Recent Activity" into sessions. One row represents one session, showing time, turns, processed tokens, and cached tokens together.
- The Today view now displays an hourly bar chart (0-23 hours) instead of a daily chart.
- Connected line charts with straight lines and added a compact tooltip showing token counts when hovering over data points.
- Merged "Plan usage remaining" and "Current windows" into a single "Usage windows" section, showing remaining percentage and tokens in one row per limit. Fixed the toggle for 5-hour/weekly limits to apply to both.
- Removed the redundant "Claude vs Codex" chart.
- Changed the Large widget's 7-day chart to a line chart with axis labels and fixed rendering cut-offs.
- Fixed an issue where widgets would temporarily display empty content right after startup (now falls back to the most recent data).

### System Requirements

- macOS 14.0 or later
- Apple Silicon / Intel Mac (Universal Binary)

### Installation

Unzip `TokenMeter-1.1.7.zip` and move `TokenMeter.app` to `/Applications`. If you are already using Token Meter, you can update from the menu by clicking "Check for Updates…".

The app is Developer ID signed and notarized by Apple. The update ZIP is also verified by Sparkle's EdDSA signature.

## 中文

此更新大幅改善了仪表盘和小组件的可读性，让“实际消耗的 Token”一目了然。

### 变更内容

- 将仪表盘的数值统一为“实际处理 Token”基准（排除了每轮对话重复发送的缓存部分）。概要卡片突出显示实际处理量，并并记包含内缓在内的总量。
- 将“最近活动”按会话进行聚合。一行代表一个会话，集中显示时间段、轮数、实际处理量和缓存量。
- 今日视图中用每小时（0~23时）的柱状图代替了日图表。
- 将折线图改为直线连接，鼠标悬停在各点时会显示轻量提示框以查看当天的 Token 数。
- 将“计划剩余用量” and “当前时间窗”整合为统一的“使用窗口 (Usage windows)”，按限额类型在一行中显示剩余 % 和 Token 数。修复了 5 小时/每周限制的显示开关同时作用于两者。
- 删除了重复的“Claude vs Codex”图表。
- 将大型小组件的 7 天图表更改为带有轴标签的折线图，解决了上下内容被遮挡的问题。
- 修复了刚启动等情况下小组件临时显示为空白的问题（现在会回退到最近的数据）。

### 运行环境

- macOS 14.0 或更高版本
- Apple Silicon / Intel Mac（通用二进制）

### 安装方法

解压 `TokenMeter-1.1.7.zip`，然后将 `TokenMeter.app` 移至 `/Applications`。如果您已在使用 Token Meter，可通过应用内的“Check for Updates…”进行更新。

应用已使用 Developer ID 签名并已通过 Apple 公证。更新 ZIP 包也通过 Sparkle 的 EdDSA 签名进行验证。

## 한국어

대시보드와 위젯의 가독성을 크게 개선하여 '실제 사용된 토큰'을 한눈에 확인할 수 있도록 한 업데이트입니다.

### 변경 내용

- 대시보드의 수치를 '실제 처리된 토큰' 기준으로 통일(매 턴 재전송되는 캐시 분량 제외). 요약 카드에는 실제 처리량을 크게 표시하고 캐시를 포함한 총량을 병기합니다.
- '최근 활동'을 세션 단위로 집계. 1행 = 1세션으로 시간대, 턴 수, 실제 처리량, 캐시량을 모아서 표시합니다.
- 오늘(Today) 보기에서는 일별 그래프 대신 시간별(0~23시) 막대 그래프를 표시합니다.
- 꺾은선 그래프를 직선으로 연결하고 각 포인트에 커서를 올리면 해당 일의 토큰 수를 툴팁으로 띄워줍니다.
- 'Plan usage remaining'과 'Current windows'를 'Usage windows'로 통합하여 제한 영역별 잔여 %와 토큰 수를 한 행에 표시합니다. 5시간/주간 제한 표시 토글이 양쪽 모두에 적용되도록 수정했습니다.
- 중복되던 'Claude vs Codex' 그래프를 삭제했습니다.
- 대형 위젯의 7일 그래프를 축 레이블이 있는 꺾은선으로 변경하여 상하단 잘림 현상을 해결했습니다.
- 앱 실행 직후 등에 위젯이 일시적으로 빈 화면으로 표시되던 문제를 수정했습니다(최근 데이터로 폴백).

### 작동 환경

- macOS 14.0 이상
- Apple Silicon / Intel Mac (Universal Binary)

### 설치 방법

`TokenMeter-1.1.7.zip` 압축을 풀고 `TokenMeter.app`을 `/Applications`로 이동하십시오. 이미 Token Meter를 사용 중인 경우 앱 내 'Check for Updates…'를 통해 업데이트할 수 있습니다.

앱은 Developer ID로 서명되었으며 Apple의 공증(Notarization)을 받았습니다. 업데이트용 ZIP은 Sparkle의 EdDSA 서명으로도 검증됩니다.
