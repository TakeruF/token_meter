# Token Meter 1.1.7

ダッシュボードとWidgetの見やすさを大きく改善し、「実際に使ったトークン」が一目で分かるようにするアップデートです。

## 変更内容

- ダッシュボードの数値を「実処理トークン」基準に統一（毎ターン再送されるキャッシュ分を除外）。Summaryカードは実処理量を大きく表示し、キャッシュ込みの総量を併記
- 「最近のアクティビティ」をセッション単位に集約。1行＝1セッションで、時間帯・ターン数・実処理量・キャッシュ量をまとめて表示
- Today表示では日別グラフの代わりに時間別（0〜23時）の棒グラフを表示
- 折れ線グラフを直線で接続し、各点にカーソルを合わせるとその日のトークン数をコンパクトなツールチップで表示
- 「Plan usage remaining」と「Current windows」を1つの「Usage windows」に統合し、枠ごとに残量%とトークン数を1行で表示。5時間枠／週次枠の表示トグルが両方に効くよう修正
- 重複していた「Claude vs Codex」グラフを削除
- Largeウィジェットの7日間グラフを軸ラベル付きの折れ線に変更し、上下の見切れを解消
- 起動直後などにウィジェットが一時的に空表示へ落ちる問題を修正（直近のデータにフォールバック）

## 動作環境

- macOS 14.0以降
- Apple Silicon / Intel Mac（Universal Binary）

## インストール

`TokenMeter-1.1.7.zip`を展開し、`TokenMeter.app`を`/Applications`へ移動してください。既にToken Meterを使用している場合は、アプリ内の「Check for Updates…」から更新できます。

アプリはDeveloper IDで署名し、AppleのNotary Serviceで公証しています。更新ZIPはSparkleのEdDSA署名でも検証されます。
