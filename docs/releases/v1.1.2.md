# Token Meter 1.1.2

メニューバーのCompact表示を、Claude／Codexそれぞれのロゴと短い値のペアとして読みやすくするパッチアップデートです。

## 変更内容

- ClaudeとCodexが有効な場合、`Claudeロゴ 68%  Codexロゴ 42%`の形式で表示
- Compact表示からプロバイダー間の`·`を削除
- Compact表示の割合から`left`を削除
- ロゴと値を固定ペアにして、メニューバー上で分離しないよう改善
- 割合が取得できない場合は、従来どおりトークン数または`—`を表示
- オンボーディングのCompactプレビューを実際の表示と統一

## 動作環境

- macOS 14.0以降
- Apple Silicon / Intel Mac（Universal Binary）

## インストール

`TokenMeter-1.1.2.zip`を展開し、`TokenMeter.app`を`/Applications`へ移動してください。既にToken Meterを使用している場合は、アプリ内の「Check for Updates…」から更新できます。

アプリはDeveloper IDで署名し、AppleのNotary Serviceで公証しています。更新ZIPはSparkleのEdDSA署名でも検証されます。
