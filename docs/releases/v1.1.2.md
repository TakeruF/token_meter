# Token Meter 1.1.2

## English

This patch update improves menu bar readability in Compact mode by displaying each provider logo alongside a shortened usage value.

### Changes

- Shows both Claude and Codex when enabled in the format of `[Claude Logo] 68%  [Codex Logo] 42%`.
- Removed the separator dot (`·`) between providers in Compact mode.
- Removed the word "left" from usage percentages in Compact mode.
- Grouped logos and values as fixed pairs to prevent them from splitting across the menu bar.
- Falls back to raw token counts or `—` when remaining percentage is unavailable.
- Unified the onboarding Compact preview with actual menu bar rendering.

### System Requirements

- macOS 14.0 or later
- Apple Silicon / Intel Mac (Universal Binary)

### Installation

Unzip `TokenMeter-1.1.2.zip` and move `TokenMeter.app` to `/Applications`. If you are already using Token Meter, you can update from the menu by clicking "Check for Updates…".

The app is Developer ID signed and notarized by Apple. The update ZIP is also verified by Sparkle's EdDSA signature.

## 한국어

메뉴 막대의 컴팩트(Compact) 표시를 Claude 및 Codex 로고와 짧은 수치 쌍으로 조합하여 가독성을 높인 패치 업데이트입니다.

### 변경 내용

- 두 제공자가 활성화된 경우 `[Claude 로고] 68%  [Codex 로고] 42%` 형식으로 표시.
- 컴팩트 표시 모드에서 제공자 간 구분 기호인 점(`·`) 삭제.
- 컴팩트 표시의 백분율 텍스트에서 'left' 문구 삭제.
- 로고와 값을 고정 쌍으로 묶어 메뉴 막대 위에서 흩어지지 않도록 개선.
- 백분율을 가져올 수 없는 경우 기존과 동일하게 토큰 수 또는 `—` 표시.
- 온보딩 가이드 내 컴팩트 미리보기를 실제 메뉴 막대 표시와 통일.

### 작동 환경

- macOS 14.0 이상
- Apple Silicon / Intel Mac (Universal Binary)

### 설치 방법

`TokenMeter-1.1.2.zip` 압축을 풀고 `TokenMeter.app`을 `/Applications`로 이동하십시오. 이미 Token Meter를 사용 중인 경우 앱 내 'Check for Updates…'를 통해 업데이트할 수 있습니다.

앱은 Developer ID로 서명되었으며 Apple의 공증(Notarization)을 받았습니다. 업데이트용 ZIP은 Sparkle의 EdDSA 서명으로도 검증됩니다.
