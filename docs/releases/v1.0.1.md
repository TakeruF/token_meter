# Token Meter 1.0.1

## English

This update improves the distinction between Claude and Codex and enhances macOS widget displays.

### Changes

- Added official logos for Claude and Codex to the Dashboard, menu bar, Settings, Setup, and widgets.
- Enabled automatic inversion for the Claude icon to match system light/dark modes.
- Applied dedicated light/dark assets for the Codex icon.
- Added Claude's 5-hour reset time to the 2x2 widget.
- Redesigned the 2x4 widget into a two-column layout for Claude and Codex to fix a bottom truncation issue.

### System Requirements

- macOS 14.0 or later
- Apple Silicon / Intel Mac (Universal Binary)

### Installation

Unzip `TokenMeter-1.0.1.zip` and move `TokenMeter.app` to `/Applications`. If you are already using Token Meter, you can update from the menu by clicking "Check for Updates…".

The app is Developer ID signed and notarized by Apple. The update ZIP is also verified by Sparkle's EdDSA signature.
