# Token Meter 1.0.1

ClaudeとCodexの識別性を高め、macOSウィジェットの表示を改善するアップデートです。

## 変更内容

- ClaudeとCodexの公式マークをDashboard、メニューバー、Settings、Setup、Widgetへ追加
- Claudeアイコンをシステムのライト／ダークモードに合わせて自動反転
- Codexアイコンにライト／ダーク専用画像を適用
- 2×2 WidgetへClaudeの5時間リセット時刻を追加
- 2×4 WidgetをClaude／Codexの2列レイアウトへ変更し、下部が見切れる問題を修正

## 動作環境

- macOS 14.0以降
- Apple Silicon / Intel Mac（Universal Binary）

## インストール

`TokenMeter-1.0.1.zip`を展開し、`TokenMeter.app`を`/Applications`へ移動してください。既にToken Meter 1.0.0を使用している場合は、アプリ内の「Check for Updates…」から更新できます。

アプリはDeveloper IDで署名し、AppleのNotary Serviceで公証しています。更新ZIPはSparkleのEdDSA署名でも検証されます。
