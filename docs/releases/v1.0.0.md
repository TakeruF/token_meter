# Token Meter 1.0.0

Token Meterの最初の公開リリースです。ClaudeとCodexの利用状況を、macOSのメニューバー、Dashboard、デスクトップウィジェットから確認できます。

## 主な機能

- ClaudeとCodexのローカルログから、入力・キャッシュ・出力トークン、モデル別・日別の利用状況を集計
- Claude OAuth連携を有効にした場合、5時間・7日間の利用率とリセット時刻を表示
- Codexが報告する5時間・週次の利用率、残量、リセット時刻を表示
- Small、Medium、LargeのmacOSウィジェットに対応
- DashboardとSettingsを同じウィンドウのタブとして表示
- メニューバーのMeterアイコンを設定から表示・非表示に切り替え
- Sparkleによるアプリ内更新確認、ダウンロード、インストールに対応

## 改善・修正

- DashboardまたはSettingsを開いたとき、既存ウィンドウを最前面へ表示
- Settingsを歯車アイコンへ変更
- Claude Codeの利用枠がClaude全体で共有されることに合わせ、UI表記を「Claude」へ変更
- 本体アプリが作成したApp Groupスナップショットをウィジェットが確実に読み込めるよう修正
- データ未作成時のウィジェット再読み込み間隔を短縮

## 動作環境

- macOS 14.0以降
- Apple Silicon / Intel Mac（Universal Binary）

## インストール

1. `TokenMeter-1.0.0.zip`をダウンロードして展開します。
2. `TokenMeter.app`を`/Applications`へ移動します。
3. Token Meterを起動します。

アプリはDeveloper IDで署名し、AppleのNotary Serviceで公証しています。以降のリリースはアプリ内の「Check for Updates…」から確認できます。

## プライバシー

トークン履歴の集計と保存はローカルで行います。プロンプト本文、応答本文、認証情報は保存しません。Claude OAuth使用量照会を明示的に有効にした場合のみAnthropicへ接続し、更新確認時はGitHub上の更新フィードへ接続します。

詳細は[プライバシーポリシー](../privacy.md)と[データソースの説明](../data-sources.md)を参照してください。
