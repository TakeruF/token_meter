# Token Meter 1.0.0

## English

First public release of Token Meter. Check your Claude and Codex usage directly from the macOS menu bar, Dashboard, and desktop widgets.

### Key Features

- Aggregates input, cached, and output tokens from local logs, grouped by model and day.
- Displays remaining usage percentage and reset times for 5-hour/7-day windows when Claude OAuth integration is enabled.
- Displays 5-hour and weekly usage rates, remaining amounts, and reset times reported by Codex.
- Supports Small, Medium, and Large macOS widgets.
- Hosts Dashboard and Settings in tabs within the same window.
- Allows toggling the menu bar Meter icon in Settings.
- Supports in-app updates, downloads, and installation powered by Sparkle.

### Enhancements & Fixes

- Brings existing windows to the front when opening Dashboard or Settings.
- Replaced the settings tab icon with a gear.
- Updated UI text to "Claude" as Claude Code's limits are shared across Claude.
- Fixed widget loading to ensure it reads the App Group snapshot created by the main app.
- Shortened widget refresh intervals when data has not been generated yet.

### System Requirements

- macOS 14.0 or later
- Apple Silicon / Intel Mac (Universal Binary)

### Installation

1. Download and unzip `TokenMeter-1.0.0.zip`.
2. Move `TokenMeter.app` to `/Applications`.
3. Open Token Meter.

The app is Developer ID signed and notarized by Apple. Subsequent updates can be installed via "Check for Updates..." in the app.

### Privacy

Token history aggregation and storage are fully local. The app never saves prompt text, response bodies, or credentials. It connects to Anthropic only when Claude OAuth usage tracking is explicitly enabled, and accesses the GitHub release feed solely to check for updates.
For more details, see the [Privacy Policy](../privacy.md) and [Data Sources](../data-sources.md).

详情请参阅 [隐私政策](../privacy.md) 与 [数据源说明](../data-sources.md)。

## 한국어

Token Meter의 첫 번째 공식 릴리스입니다. Claude와 Codex의 사용량을 macOS 메뉴 막대, 대시보드(Dashboard), 데스크톱 위젯에서 모니터링할 수 있습니다.

### 주요 기능

- 로컬 로그로부터 입력, 캐시, 출력 토큰 양을 분석하여 모델별 및 일별 사용량 집계.
- Claude OAuth 연동 활성화 시, 5시간 및 7일 주기 사용량 비율과 초기화 시각 표시.
- Codex가 보고하는 5시간 및 주간 사용률, 잔여량, 초기화 시각 표시.
- Small, Medium, Large 크기의 macOS 위젯 지원.
- 대시보드와 설정을 같은 창의 탭 구조로 제공.
- 설정에서 메뉴 막대 계기판(Meter) 아이콘 노출 여부 선택 가능.
- Sparkle 기반의 앱 내 업데이트 확인, 다운로드 및 설치 기능 제공.

### 개선 및 수정 사항

- 대시보드나 설정을 열 때 기존 창을 화면 최전면으로 띄우도록 수정.
- 설정 탭 아이콘을 톱니바퀴 형태로 변경.
- Claude Code의 사용 한도가 Claude 전체와 공유되는 점을 반영하여 UI 표기를 'Claude'로 변경.
- 앱 본체에서 생성한 App Group 스냅샷 데이터를 위젯이 확실하게 읽어가도록 수정.
- 수집된 데이터가 아직 없을 때 위젯의 갱신 대기 시간 단축.

### 작동 환경

- macOS 14.0 이상
- Apple Silicon / Intel Mac (Universal Binary)

### 설치 방법

1. `TokenMeter-1.0.0.zip` 다운로드 후 압축 해제.
