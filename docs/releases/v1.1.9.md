# Token Meter 1.1.9

初期設定とダッシュボードを使いやすくするアップデートです。

## 変更内容

- 初期設定に、選択したプロバイダーのコマンドラインツールが未インストールの場合のインストール手順画面を追加。インストール済みなら自動でスキップ
- 設定画面に「セットアップガイドをもう一度実行」ボタンを追加し、初期設定フローをいつでも再表示できるように
- ダッシュボードの「今日の時間別推移」を折れ線グラフに変更し、時間帯ごとの利用の流れを見やすく表示。ホバーで各プロバイダーのトークン数を確認可能

## 動作環境

- macOS 14.0以降
- Apple Silicon / Intel Mac（Universal Binary）

## インストール

`TokenMeter-1.1.9.zip`を展開し、`TokenMeter.app`を`/Applications`へ移動してください。既にToken Meterを使用している場合は、アプリ内の「Check for Updates…」から更新できます。

アプリはDeveloper IDで署名し、AppleのNotary Serviceで公証しています。更新ZIPはSparkleのEdDSA署名でも検証されます。
