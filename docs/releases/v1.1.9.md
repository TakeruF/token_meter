# Token Meter 1.1.9

## English

This update improves the usability of the setup guide and dashboard.

### Changes

- Added an installation steps screen to the setup guide when command-line tools for the selected provider are not installed. Skips automatically if already installed.
- Added a "Run setup guide again" button to Settings to let you reopen the onboarding flow anytime.
- Changed the "Hourly Trend Today" on the dashboard to a line chart to make usage patterns clearer. Hover to view token counts for each provider.

### System Requirements

- macOS 14.0 or later
- Apple Silicon / Intel Mac (Universal Binary)

### Installation

Unzip `TokenMeter-1.1.9.zip` and move `TokenMeter.app` to `/Applications`. If you are already using Token Meter, you can update from the menu by clicking "Check for Updates…".

The app is Developer ID signed and notarized by Apple. The update ZIP is also verified by Sparkle's EdDSA signature.

## 中文

此更新提高了初始设置和仪表盘的易用性。

- 在初始设置中，若所选提供商的命令行工具未安装，则添加安装步骤界面。若已安装则自动跳过。

- macOS 14.0 或更高版本

## 한국어

초기 설정 및 대시보드 사용성을 향상시키는 업데이트입니다.

### 변경 내용

- 초기 설정에 선택한 제공자의 명령줄 도구가 설치되지 않은 경우의 설치 단계 화면을 추가. 이미 설치된 경우 자동으로 건너뜀.
- 설정 화면에 '설정 가이드 다시 실행' 버튼을 추가하여 초기 설정 흐름을 언제든지 다시 표시할 수 있도록 개선.
- 대시보드의 '오늘의 시간별 추이'를 꺾은선 그래프로 변경하여 시간대별 사용 흐름을 보기 쉽게 표시. 마우스 호버 시 각 제공자의 토큰 수 확인 가능.

### 작동 환경

- macOS 14.0 이상
- Apple Silicon / Intel Mac (Universal Binary)

### 설치 방법

`TokenMeter-1.1.9.zip` 압축을 풀고 `TokenMeter.app`을 `/Applications`로 이동하십시오. 이미 Token Meter를 사용 중인 경우 앱 내 'Check for Updates…'를 통해 업데이트할 수 있습니다.

앱은 Developer ID로 서명되었으며 Apple의 공증(Notarization)을 받았습니다. 업데이트용 ZIP은 Sparkle의 EdDSA 서명으로도 검증됩니다.
