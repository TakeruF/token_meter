# Token Meter 1.1.1

メニューバーのCompact表示におけるブランドロゴの描画サイズを修正するパッチアップデートです。

## 変更内容

- Claude／CodexロゴをStatus Item専用のAppKit画像として描画
- 512px画像のintrinsic sizeがメニューバーへ伝播しないよう、論理サイズを事前に固定
- Claude／CodexロゴをmacOSのtemplate画像として扱い、ライト／ダークモードと選択状態へ追従
- Settings、Setup、オンボーディングのプレビューを実際のメニューバー描画と統一

## 動作環境

- macOS 14.0以降
- Apple Silicon / Intel Mac（Universal Binary）

## インストール

`TokenMeter-1.1.1.zip`を展開し、`TokenMeter.app`を`/Applications`へ移動してください。既にToken Meterを使用している場合は、アプリ内の「Check for Updates…」から更新できます。

アプリはDeveloper IDで署名し、AppleのNotary Serviceで公証しています。更新ZIPはSparkleのEdDSA署名でも検証されます。
