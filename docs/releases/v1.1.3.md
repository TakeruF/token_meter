# Token Meter 1.1.3

## English

This update aligns Codex remaining usage with correct account-level limits and improves menu bar and widget displays. It also introduces the official multi-language About page.

### Changes

- Fixed an issue where model-specific Codex limits were incorrectly treated as account-wide limits.
- Repaired incorrect Codex remaining usage data stored in previous versions upon application startup.
- Fixed a layout bug in Compact mode where Claude and Codex could not be shown simultaneously.
- Changed the widget's reset label to "5h limit".
- Added the official About page containing screenshots and widget showcases.
- Enabled support for English, Japanese, Chinese, and Korean on the About page.

### System Requirements

- macOS 14.0 or later
- Apple Silicon / Intel Mac (Universal Binary)

### Installation

Unzip `TokenMeter-1.1.3.zip` and move `TokenMeter.app` to `/Applications`. If you are already using Token Meter, you can update from the menu by clicking "Check for Updates…".

The app is Developer ID signed and notarized by Apple. The update ZIP is also verified by Sparkle's EdDSA signature.

## 한국어

Codex 잔여량 표시를 올바른 계정 기준 한도로 맞추고 메뉴 막대 및 위젯 표시를 개선하는 업데이트입니다. 더불어 4개 언어를 지원하는 공식 소개(About) 페이지를 추가했습니다.

### 변경 내용

- 모델별 Codex 사용 한도를 계정 전체 한도로 잘못 처리하던 오류 수정.
- 이전 버전에서 잘못 저장된 Codex 잔여량 데이터를 실행 시 올바른 값으로 복구하는 로직 추가.
- 컴팩트(Compact) 표시 모드에서 Claude와 Codex가 동시에 표시되지 않던 문제 해결.
- 위젯의 초기화 표시 문구를 '5h limit'로 변경.
- 스크린샷과 위젯 소개가 포함된 공식 소개 페이지 추가.
- 소개 페이지에 한국어, 영어, 중국어, 일본어 지원 추가.

### 작동 환경

- macOS 14.0 이상
- Apple Silicon / Intel Mac (Universal Binary)

### 설치 방법

`TokenMeter-1.1.3.zip` 압축을 풀고 `TokenMeter.app`을 `/Applications`로 이동하십시오. 이미 Token Meter를 사용 중인 경우 앱 내 'Check for Updates…'를 통해 업데이트할 수 있습니다.

앱은 Developer ID로 서명되었으며 Apple의 공증(Notarization)을 받았습니다. 업데이트용 ZIP은 Sparkle의 EdDSA 서명으로도 검증됩니다.
