# Token Meter 1.1.3

Codexの残量表示を正しいアカウント利用枠へ揃え、メニューバーとWidgetの表示を改善するアップデートです。あわせて、4言語対応の公式Aboutページを追加しました。

## 変更内容

- Codexのモデル固有利用枠をアカウント全体の利用枠として扱わないよう修正
- 以前のバージョンで保存された誤ったCodex残量を、起動時に正しい値へ修復
- Compact表示でClaudeとCodexを同時に表示できない問題を修正
- Widgetのリセット表示を`5h limit`表記へ変更
- スクリーンショットとWidget紹介を含む公式Aboutページを追加
- Aboutページを日本語・英語・中国語・韓国語に対応

## 動作環境

- macOS 14.0以降
- Apple Silicon / Intel Mac（Universal Binary）

## インストール

`TokenMeter-1.1.3.zip`を展開し、`TokenMeter.app`を`/Applications`へ移動してください。既にToken Meterを使用している場合は、アプリ内の「Check for Updates…」から更新できます。

アプリはDeveloper IDで署名し、AppleのNotary Serviceで公証しています。更新ZIPはSparkleのEdDSA署名でも検証されます。
