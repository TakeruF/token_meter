# Token Meter 1.2.4

## English

This update lets you choose the widget background style and makes the widget look right in the system's tinted (accented) rendering mode.

### Changes

- Added a widget background setting: keep the existing **Solid** background, or switch to **Clear (Liquid Glass)** for a translucent glass look. Clear is offered only on macOS 26 and later, where the Liquid Glass API exists; on older systems the widget stays Solid. The choice is stored alongside your existing language and notation settings and the widget refreshes as soon as you change it.
- Optimized the widget for the system's tinted (accented) appearance: brand icons, the progress-bar fill, the headline percentage, and the large-widget chart line now take the accent colour while labels stay legible, so the light, dark, tinted, and clear widget appearances all render cleanly.

### System Requirements

- macOS 14.0 or later (Clear / Liquid Glass background requires macOS 26 or later)
- Apple Silicon / Intel Mac (Universal Binary)

### Installation

Unzip `TokenMeter-1.2.4.zip` and move `TokenMeter.app` to `/Applications`. If you are already using Token Meter, you can update from the menu by clicking "Check for Updates…".

The app is Developer ID signed and notarized by Apple. The update ZIP is also verified by Sparkle's EdDSA signature.

### 動作環境

- macOS 14.0以降（クリア／Liquid Glass背景はmacOS 26以降が必要）

## 中文

此更新让您可以选择小组件的背景样式，并针对系统的着色（强调色）渲染模式优化了小组件的显示效果。

### 变更内容

- 新增了小组件背景设置：可以保留现有的**实心**背景，或切换到**透明（Liquid Glass）**以获得半透明的玻璃质感。透明样式仅在提供 Liquid Glass API 的 macOS 26 及更高版本上可选，较旧的系统则保持实心显示。该选项与您现有的语言和数字表示设置一起保存，更改后小组件会立即刷新。
- 针对系统的着色（强调色）外观优化了小组件：品牌图标、进度条填充、顶部百分比以及大尺寸小组件的图表线条现在会采用强调色，而标签保持清晰可读，因此浅色、深色、着色和透明外观都能干净地呈现。

### 运行环境

- macOS 14.0 或更高版本（透明 / Liquid Glass 背景需要 macOS 26 或更高版本）
- Apple Silicon / Intel Mac（通用二进制）

### 安装方法

解压 `TokenMeter-1.2.4.zip`，然后将 `TokenMeter.app` 移至 `/Applications`。如果您已在使用 Token Meter，可通过应用内的“Check for Updates…”进行更新。

应用已使用 Developer ID 签名并已通过 Apple 公证。更新 ZIP 包也通过 Sparkle 的 EdDSA 签名进行验证。

## 한국어

위젯 배경 스타일을 선택할 수 있게 하고, 시스템의 틴트(강조색) 렌더링 모드에서도 위젯이 올바르게 보이도록 최적화한 업데이트입니다.

### 변경 내용

- 위젯 배경 설정을 추가했습니다. 기존의 **솔리드** 배경을 유지하거나, 반투명 유리 느낌의 **클리어(Liquid Glass)**로 전환할 수 있습니다. 클리어는 Liquid Glass API가 있는 macOS 26 이상에서만 선택할 수 있으며, 이전 시스템에서는 솔리드로 표시됩니다. 이 선택은 기존 언어 및 표기 설정과 함께 저장되며, 변경하면 위젯에 즉시 반영됩니다.
- 시스템의 틴트(강조색) 표시에 맞춰 위젯을 최적화했습니다. 브랜드 아이콘, 진행 막대 채움, 상단 백분율, 큰 위젯의 차트 선이 강조색을 받고 레이블은 읽기 쉽게 유지되므로 라이트, 다크, 틴트, 클리어 모든 표시에서 깔끔하게 렌더링됩니다.

### 작동 환경

- macOS 14.0 이상 (클리어 / Liquid Glass 배경은 macOS 26 이상 필요)
- Apple Silicon / Intel Mac (Universal Binary)

### 설치 방법

`TokenMeter-1.2.4.zip` 압축을 풀고 `TokenMeter.app`을 `/Applications`로 이동하십시오. 이미 Token Meter를 사용 중인 경우 앱 내 'Check for Updates…'를 통해 업데이트할 수 있습니다.

앱은 Developer ID로 서명되었으며 Apple의 공증(Notarization)을 받았습니다. 업데이트용 ZIP은 Sparkle의 EdDSA 서명으로도 검증됩니다.
